# LaunchDeck

LaunchDeckは、Novation Launchpad Mini MK3 を「Stream Deckのようなショートカットパッド」として使用するためのツールです。

アプリを起動している間はショートカットパッドとして動作し、終了すると通常のLaunchpadとして使用できます。

## インストール方法

`settings.bat` を実行してください。  
必要な環境構築および初期設定が行われます。

---

## ビルド方法（exe化）

`build.bat` を実行してください。  
exeファイルが生成されます。

---

## 自動起動について

アプリ内の「自動起動」スイッチで切り替えできます。

- ON：Windows起動時に自動で起動
- OFF：自動起動を無効化

内部的には以下のフォルダにショートカットを作成・削除しています：

%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup

---

## トレイ常駐について

ウィンドウを閉じる際、アプリの動作を「終了」または「バックグラウンド継続」から選択できます。

「バックグラウンド継続」を選択した場合、アプリは終了せずシステムトレイへ移動します。  
そのままバックグラウンドで動作を継続できます。
